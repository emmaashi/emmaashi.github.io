nightlight

Addressing and ensuring the safety of women+ when taking public transport at night.

The inspiration behind ‘nightlight’ stemmed from the fear of taking public transportation systems, like the TTC, alone.

  • 🥇 1st Place Waterloo TechNova Best Connectivity Hack
  • 🥇 People's Choice Award
  • Prototyped using Figma
  • Developed using TailwindCSS, a utility-first CSS framework. HTML, CSS, and JavaScript was used for the front-end.
